# 一、产品定位（一句话定生死）

**这是一个：
"当我犹豫、退缩、想逃的时候，用一句毛选逼我站直的工具。"**

不是学习，不是收藏，不是传播。

---

# 二、功能结构总览（极简，但有锋芒）

```
App
├── 首页（核心单页流程）
│   ├── 空闲状态：显示最近一次记录
│   ├── 选择当前处境
│   ├── 抽取并展示语录
│   ├── 行动确认
│   └── 完成记录 → 自动返回
│
├── 我的记录（Sheet）
│   ├── 时间线列表
│   ├── 处境 + 语录 + 选择
│   └── 点击查看详情
│
├── 语录池（Sheet，只读）
│   ├── 按"处境"分类（emoji 切换）
│   ├── 左右滑动切换分类
│   └── 原文 + 来源
│
└── Widget
    ├── 桌面小组件（小/中/大）
    └── 锁屏小组件（圆形/矩形/内联）
```

**没有登录、没有云端、没有社交。**

---

# 三、核心交互：单页状态流程（主路径）

这是整个 App **唯一被精心设计的路径**，所有交互在单页完成。

---

## 🟡 Phase 1｜idle（空闲状态）

```
--------------------------------
|                              |
|    最近：01.07 · 🤔 犹豫      |
|   "没有调查，就没有发言权"    |
|        ✅ 照做了              |
|                              |
|          给我一句            |
|     现在能站住的话            |
|                              |
|        [ 开始 ]               |
|                              |
--------------------------------
```

* 顶部显示最近一次记录（如有）
* 用户一进来就知道：这是个**工具，不是内容 App**
* 按下开始，振动反馈

---

## 🟡 Phase 2｜selectingSituation（选择处境）

```
--------------------------------
|                              |
|  你现在的状态是？             |
|                              |
|  ○ 🤔 我在犹豫                |
|  ○ 😰 我在退缩                |
|  ○ 💪 我在硬撑                |
|  ○ 😔 我在被动挨打            |
|                              |
|        [ 确认 ]               |
--------------------------------
```

**关键点：**

* 不是情绪（开心/难过）
* 是**行动前状态**
* 选择时有触觉反馈
* 这是毛选真正工作的地方

---

## 🟡 Phase 3｜showingQuote（语录呈现）

```
--------------------------------
|                              |
|      🤔 我在犹豫              |
|                              |
|  "没有调查，                 |
|   就没有发言权。"            |
|                              |
|        ——《反对本本主义》      |
|                              |
|        [ 下一步 ]             |
--------------------------------
```

**设计原则：**

* 不解释
* 不高亮
* 不做金句海报
* 字体偏正文，不煽情
* 顶部保留已选处境

> 这是"迎面撞上"，不是"被喂"。

---

## 🟡 Phase 4｜selectingAction（行动确认）

```
--------------------------------
|                              |
|      🤔 我在犹豫              |
|  "没有调查，就没有发言权"    |
|      ——《反对本本主义》       |
|                              |
|  面对这句话                   |
|                              |
|  ○ ✅ 我照做了                |
|  ○ ❌ 我没照做                |
|  ○ 🏃 我逃避了                |
|                              |
|        [ 记录 ]               |
--------------------------------
```

* 不给"感受"
* 不给"是否有帮助"
* 只问：**你站哪边**
* 顶部保留处境 + 语录

---

## 🟡 Phase 5｜completed（完成）

```
--------------------------------
|                              |
|      🤔 我在犹豫              |
|  "没有调查，就没有发言权"    |
|      ——《反对本本主义》       |
|        ✅ 我照做了            |
|                              |
|          已记录               |
|                              |
|      去做你的事。             |
|                              |
--------------------------------
```

**2秒后自动返回首页，不会推荐下一句，不会诱导继续使用。**

# 四、我的记录（残酷回看）

这是你**半年后会感谢自己的地方**。

---

## 时间线视图

```
--------------------------------
|  2025.03.18 12:34  🤔 犹豫   |
|  "没有调查，就没有发言权"    |
|   → 🏃 逃避了                 |
|                              |
|  2025.02.04 09:15  💪 硬撑   |
|  "革命不是请客吃饭"          |
|   → ✅ 照做了                 |
--------------------------------
```

* 不做统计
* 不给总结
* 只是**事实陈列**

---

## 点击某条记录

```
--------------------------------
|  状态：🤔 我在犹豫            |
|  时间：2025.03.18            |
|                              |
|  "没有调查，                 |
|   就没有发言权。"            |
|        ——《反对本本主义》     |
|                              |
|  我的选择：🏃 逃避了          |
|                              |
--------------------------------
```

**没有"后悔按钮"。**

# 五、语录池设计（为你兜底）

这是防止用户觉得"是不是就这几句"的地方。

```
语录池（emoji 标签切换 + 左右滑动）
├── 🤔 犹豫时该看的（5条）
│   ├── 实践 / 调查
│   └── 判断与发言
│
├── 😰 退缩时该看的（5条）
│   ├── 斗争 / 代价
│   └── 主动性
│
├── 💪 硬撑时该看的（5条）
│   ├── 长期战
│   └── 意志
│
└── 😔 被动挨打时该看的（5条）
    ├── 敌我关系
    └── 现实判断
```

**只读，不可收藏，不可点赞。**

# 六、交互流程图（状态机版）

```
启动 App
   ↓
idle（空闲，显示最近记录）
   ↓ [开始]
selectingSituation（选择处境）
   ↓ [确认]
showingQuote（展示语录）
   ↓ [下一步]
selectingAction（行动确认）
   ↓ [记录]
completed（完成）
   ↓ [2秒自动]
idle（回到首页）
```

这是一个**天然防沉迷结构**。

# 七、Widget 功能

## 桌面小组件

| 尺寸 | 内容 |
|------|------|
| 小 | 行动状态 + 语录 |
| 中 | 行动状态 + 日期 + 语录 + 出处 |
| 大 | 完整展示 |

## 锁屏小组件

| 样式 | 内容 |
|------|------|
| 圆形 | 状态图标 + 文字 |
| 矩形 | 语录 + 状态 |
| 内联 | 一行提醒 |

## 状态颜色

| 选择 | 颜色 |
|------|------|
| ✅ 照做了 | 🟢 绿色 |
| ❌ 没照做 | 🟠 橙色 |
| 🏃 逃避了 | 🔴 红色 |

# 八、产品底线（请你认真看）

* 这个 App **不该让人感觉舒服**
* 它应该像一个**沉默、但站得很直的人**
* 它存在的意义，是在你想逃的时候，
  **不给你台阶**
