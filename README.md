# Instructor - 站直工具

> "当我犹豫、退缩、想逃的时候，用一句毛选逼我站直的工具。"

## 📱 这是什么？

这不是学习 App，不是收藏工具，不是传播平台。

这是一个让你在想逃的时候**站直了**的工具。

---

## ⚡️ 快速开始

```bash
# 克隆项目
cd /Users/edge/side/instructor

# 打开项目
open instructor.xcodeproj

# 选择 instructor target，运行
```

---

## 🎯 核心流程

```
打开 App
   ↓
选择你的状态（犹豫/退缩/硬撑/被动挨打）
   ↓
抽一句话
   ↓
做选择：照做了 / 没照做 / 逃避了
   ↓
记录
   ↓
退出，去做你的事
```

**5 步，2 分钟，天然防沉迷。**

---

## 📖 功能

### 1. 抽一句（核心）
- 选择当前处境
- 随机抽取语录
- 行动确认

### 2. 我的记录
- 时间线回看
- 事实陈列
- 没有后悔按钮

### 3. 语录池
- 按处境分类
- 只读，不可收藏
- 原文 + 出处

### 4. 设置
- 使用说明
- 数据本地化
- 清空记录

---

## 🎨 设计原则

- **极简** - 没有 logo，没有多余文字
- **冷处理** - 不煽情，不解释，不美化
- **无社交** - 没有登录，没有云端，没有分享
- **本地化** - 所有数据本地存储
- **防沉迷** - 完成后立刻退出

## 📝 产品底线

这个 App **不该让人感觉舒服**。

它应该像一个**沉默、但站得很直的人**。

它存在的意义，是在你想逃的时候，**不给你台阶**。
